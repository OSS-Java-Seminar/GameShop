### Description

Application that offers cheap PC, PlayStation and XBox game keys.

<br />

<a name="Features"/>

### Features

- Login and register system
  - as guest
    - product overview
  - as user
    - product overview
    - adding products to shopping cart
    - purchase products
    - earn loyalty points
    - register with email
    - log in with username and password
  - as admin
    - add/remove content
    - user management
- Search bar
  - simplified way to find desired product
- Shopping cart
- Loyalty point system (users only)
  - Earning points
    - purchasing products
  - Spending points
    - discounts
    - game keys
- Multiple payment methods
  - Credit cards
  - PayPal
  - Paysafe
